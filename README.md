# 📊 Gen Z Social Media Behavior Analysis

A data science project that explores the factors influencing Generation Z's social media posting behavior through data analysis and machine learning. This project includes data preprocessing, exploratory data analysis (EDA), feature engineering, and predictive modeling to identify patterns that affect users' willingness to post on social media.

---

## 📌 Project Overview

Social media has become an essential part of Generation Z's daily life. However, despite being highly active online, many Gen Z users are becoming less willing to share content publicly. This project aims to analyze the factors behind this behavior using survey data and machine learning techniques.

---

## 🎯 Objectives

- Clean and preprocess the dataset.
- Explore relationships between variables through Exploratory Data Analysis (EDA).
- Visualize user behavior patterns.
- Build and evaluate machine learning classification models.
- Compare model performance to identify the most effective algorithm.
